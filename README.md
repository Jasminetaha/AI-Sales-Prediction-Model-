# AI-Sales-Prediction-Model-
Businesses often struggle to identify high-value customers, at-risk segments, and opportunities for targeted retention strategies. This project aims to segment customers based on purchasing behavior and provide actionable insights to support data-driven decision-making.

Tools & Technologies
- Python (Pandas, NumPy, Plotly)
- Data Cleaning & Feature Engineering
- RFM Segmentation
- Power BI (Dashboard Design)
- Git & GitHub
